## Fitur Aplikasi
- Menampilkan daftar koleksi buku perpustakaan (Bootstrap Table)
- Menampilkan spesifikasi detail buku secara dinamis berdasarkan ID
- Validasi pengaman HTTP 404 jika ID buku tidak terdaftar

---

## Cara Menjalankan Proyek Lokal
1. Clone repository ini atau download zip.
2. Pastikan XAMPP (Apache & MySQL) sudah aktif.
3. Jalankan migrasi database di terminal:
   ```bash
   php artisan migrate
4. Bukti Server Berjalan
